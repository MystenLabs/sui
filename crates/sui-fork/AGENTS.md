You are allowed to run the following commands in this crate:
- `cargo build` - to build the crate
- `cargo test -p` - to run the tests
- `cargo nextest run`
- `cargo doc --no-deps` - to generate documentation for the crate
- `cargo xclippy -D warnings` - to check for common mistakes and improve the code quality
- `cargo fmt` - to format the code according to Rust's style guidelines
