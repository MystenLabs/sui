<dd>
 Set of addresses that are banned for a given type.
 For example with <code>sui::coin::Coin</code>: If addresses A and B are banned from using
 "0...0123::my_coin::MY_COIN", this will be "0...0123::my_coin::MY_COIN" -> {A, B}.
