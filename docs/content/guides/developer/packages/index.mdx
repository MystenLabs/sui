A Move package on Sui includes one or more modules that define the package's interaction with onchain objects. You develop the logic for those modules in Move, compile them into an object, and publish that package object to a Sui network. Onchain, anyone can view a package's contents and see how its logic manipulates objects using a Sui Explorer.
