[SuiLink](https://www.suilink.io/) is a Mysten Labs product that allows you to securely link wallet addresses from different blockchain networks to your Sui wallet. When you create a SuiLink, you receive a soulbound NFT that serves as authenticated proof of ownership for your connected wallets.

## Key features

SuiLink provides the following capabilities:

- **Cross-chain identity verification**: Link wallets from [Ethereum](/concepts/sui-for-ethereum.mdx), [Solana](/concepts/sui-for-solana.mdx), and other Sui addresses to your primary Sui wallet.
- **Non-transferable proof of ownership**: Receive a [soulbound NFT](/guides/developer/nft/nft-soulbound.mdx) that cryptographically verifies wallet ownership.
- **Enhanced discoverability**: Projects in the [Sui ecosystem](https://github.com/sui-foundation/awesome-sui/tree/main) can discover you based on your activity across all connected wallets.
- **Reward eligibility**: Unlock opportunities for cross-chain rewards, priority features, and [exclusive events](https://luma.com/sui-network) within the Sui network.

### Supported networks

SuiLink currently supports the following cross-chain wallet links: Sui-to-Ethereum, Sui-to-Solana, and Sui-to-Sui.

## Creating a SuiLink

To create a SuiLink:

1. Visit [https://www.suilink.io/](https://www.suilink.io/)
2. Click **Connect Sui Wallet** to connect your Sui wallet
3. Select the blockchain network you want to link (Ethereum, Solana, or another Sui address)
4. Follow the prompts to authenticate and connect the wallet you wish to link
5. Complete the transaction to mint your SuiLink NFT

Creating a SuiLink requires a small gas fee on the Sui network to mint the NFT. Ensure you have some SUI tokens in your wallet to complete the connection.

## Managing SuiLink

You can link the same Sui wallet address to multiple wallets from Ethereum, Solana, and other Sui addresses. Each connection creates a separate SuiLink NFT in your Sui wallet.

To view your linked addresses:

1. View your SuiLink NFTs in your Sui wallet
2. Check the `network_address` field to see which wallets are linked
3. Review the timestamp to see when each link was created

You can also permanently unlink a wallet by burning the corresponding NFT pass. To unlink an address:

1. Navigate to the NFTs section of your Sui wallet
2. For the SuiLink NFT you wish to burn, select "Send" or "Transfer"
3. Enter the classic `0x0` burn address: `0x0000000000000000000000000000000000000000`

Additionally, there are various third-party tools (such as [SUI Manager](https://www.suimanager.com/burner)) that can help burn NFTs, and some wallets have a built-in burning feature.

### Querying SuiLink objects

To retrieve a user's SuiLink connections, use the following dApp Kit SDK query:

```typescript
const SUILINK_PACKAGE_ID_ETH_SOL = '0xf857fa9df5811e6df2a0240a1029d365db24b5026896776ddd1c3c70803bccd3';
const SUILINK_PACKAGE_ID_SUI = '0x73f5ab2461c5993408fff21354fa9831d4f4a66cc81382419ec29e3c80c384b5';

const SuiLink = bcs.struct('SuiLink', {
  id: bcs.struct('UID', { id: bcs.Address }),
  network_address: bcs.String,
  timestamp_ms: bcs.U64,
});

const ethSolSuiLinks = response.objects?.filter(obj => 
  obj.type?.includes(SUILINK_PACKAGE_ID_ETH_SOL)
) || [];

const suiSuiLinks = response.objects?.filter(obj => 
  obj.type?.includes(SUILINK_PACKAGE_ID_SUI)
) || [];

const parsedEthSolObjects = ethSolSuiLinks
  .filter(obj => !(obj instanceof Error))
  .map(obj => SuiLink.parse(obj.content));

const parsedSuiObjects = suiSuiLinks
  .filter(obj => !(obj instanceof Error))
  .map(obj => SuiLink.parse(obj.content));
```

This returns all SuiLink NFTs owned by the specified Sui address. Each NFT contains information about the linked wallet.

### SuiLink NFT structure

Each SuiLink NFT contains the following fields:

```typescript
{
  id: {
    id: "0xcafe"
  },
  network_address: "0xdecaf",
  timestamp_ms: "1751541273947"
}
```

The NFT type indicates which blockchain network the wallet is linked to:

```typescript
// Sui-to-Sui SuiLink (Mainnet)
0xf857fa9df5811e6df2a0240a1029d365db24b5026896776ddd1c3c70803bccd3::suilink::SuiLink<0x73f5ab2461c5993408fff21354fa9831d4f4a66cc81382419ec29e3c80c384b5::sui::Sui>

// Sui-to-Ethereum SuiLink (Testnet)
0x0025bafa2e6afa511c19bd4e95626c897e798fde629b4782fe061bdc8bd65c8a::suilink::SuiLink<0x0025bafa2e6afa511c19bd4e95626c897e798fde629b4782fe061bdc8bd65c8a::ethereum::Ethereum>

// Sui-to-Solana SuiLink (Testnet)
0x0025bafa2e6afa511c19bd4e95626c897e798fde629b4782fe061bdc8bd65c8a::suilink::SuiLink<0x0025bafa2e6afa511c19bd4e95626c897e798fde629b4782fe061bdc8bd65c8a::solana::Solana>
```

### Verifying cross-chain asset ownership

After retrieving a user's SuiLink objects, you can verify their ownership of specific assets on the linked blockchain:

1. Extract the `network_address` from the SuiLink NFT.
2. Use the appropriate blockchain API to query assets at that address. The examples use [Alchemy](https://www.alchemy.com/) for Ethereum, and [Metaplex](https://www.metaplex.com/) for Solana.
3. Verify the presence of the required asset.
4. Attribute benefits to the user through their Sui wallet.

<TabItem label="Ethereum" value="ethverify">

```typescript
// Get the user's SuiLink objects
const suilinks = await client.core.getOwnedObjects({
  owner: userSuiAddress,
  filter: { Package: SUILINK_PACKAGE_ID_ETH_SOL },
  options: { showContent: true, showType: true },
});

// Extract linked Ethereum addresses
const ethereumAddresses = suilinks.data
  .filter(obj => {
    const type = obj.data?.content?.type || '';
    return type.includes('ethereum::Ethereum');
  })
  .map(obj => obj.data?.content?.fields?.network_address)
  .filter(Boolean);

// Initialize Alchemy
const alchemy = new Alchemy({
  apiKey: alchemyApiKey,
  network: Network.ETH_MAINNET,
});

// Query Ethereum blockchain for assets
for (const ethAddress of ethereumAddresses) {
  const assets = await alchemy.nft.getNftsForOwner(ethAddress);
    
  if (assets.ownedNfts.some(nft => nft.contract.address === requiredNFT)) {
    // Grant benefit to user
    grantBenefit(userSuiAddress);
  }
}
```

```typescript
const suilinks = await client.core.getOwnedObjects({

const connection = new Connection('https://api.mainnet-beta.solana.com');
const metaplex = Metaplex.make(connection);

for (const solanaAddress of solanaAddresses) {
  const publicKey = new PublicKey(solanaAddress);
  const assets = await metaplex.nfts().findAllByOwner({ owner: publicKey });
    
  if (assets.some(nft => nft.mintAddress.toString() === requiredNFT)) {
    grantBenefit(userSuiAddress);
  }
}
```
