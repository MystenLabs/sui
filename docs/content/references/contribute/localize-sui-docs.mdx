The Sui documentation can be localized (translated) into any language of your choosing. The localization platform used is Crowdin. For more information on the localization process, see the [Crowdin introduction](https://support.crowdin.com/crowdin-intro/).

## How localization works

The Sui documentation source files are written in English. Crowdin synchronizes the English source with translation projects, where contributors can translate strings through the Crowdin editor. Translated content is then merged back into the documentation build.

## Getting started

To contribute translations:

1. Visit the [Crowdin project for Sui documentation](https://crowdin.com/) and create an account.
1. Select the language you want to translate into.
1. Choose a source file and begin translating using the Crowdin editor.
1. Submit your translations for review.

Crowdin provides translation memory and glossary features to help maintain consistency across the documentation.
