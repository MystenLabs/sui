### Installation {#installation}

To install the [React Sui Wallet Kit library from npm](https://www.npmjs.com/package/@mysten/wallet-kit), along with the Sui TypeScript SDK:

```bash
npm install @mysten/wallet-kit @mysten/sui.js
```

### Configuring the Sui Wallet Kit Provider {#configure-sui-wallet-kit-provider}

After you install the `@mysten/wallet-kit` package, you need to wrap your application with the `WalletKitProvider`.

```tsx
import { WalletKitProvider } from '@mysten/wallet-kit';

function App() {
  return (
    <WalletKitProvider>
      <YourApp />
    <WalletKitProvider />
  );
};
```

### Rendering a Connect Button {#render-connect-button}

After you wrap your app with the `WalletKitProvider`, add a `ConnectButton` component that users can click to connect their wallet to your dApp.

```tsx
import { ConnectButton } from "@mysten/wallet-kit";

export function YourApp() {
  return <ConnectButton />;
}
```
