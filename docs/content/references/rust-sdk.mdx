The Sui Rust SDK can be found in the [`sui-rust-sdk` repo](https://github.com/MystenLabs/sui-rust-sdk).

:::caution 

`sui-rust-sdk` does not support JSON RPC. If you need JSON RPC access, then the [legacy Rust SDK](https://github.com/MystenLabs/sui/tree/main/crates/sui-sdk) can be used. It is forward and backward compatible.

:::
