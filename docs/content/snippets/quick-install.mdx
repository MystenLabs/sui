Use one of the following commands for [Homebrew](https://brew.sh/) (MacOS, Linux, or Windows Subsystem for Linux) or [Chocolatey](https://chocolatey.org/) (Windows) to install Sui.

<Tabs groupId="operating-systems">

<TabItem label="Homebrew" value="mac">

```sh
$ brew install sui
```

</TabItem>

<TabItem label="Chocolatey" value="win">

```sh
$ choco install sui
```

Find more [versions of Sui for Windows](https://community.chocolatey.org/packages/sui) on the Chocolatey community website.

</TabItem>

</Tabs>
