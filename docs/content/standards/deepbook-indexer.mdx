## /get_pools

Returns a list of all available pools, each containing detailed information about the base and quote assets, as well as pool parameters like minimum size, lot size, and tick size.

### Response

Each pool object in the response includes the following fields:

-	**pool_id:** ID for the pool.
-	**pool_name:** Name of the pool.
-	**base_asset_id:** ID for the base asset.
-	**base_asset_decimals:** Number of decimals for the base asset.
-	**base_asset_symbol:** Symbol for the base asset.
-	**base_asset_name:** Name of the base asset.
-	**quote_asset_id:** ID for the quote asset.
-	**quote_asset_decimals:** Number of decimals for the quote asset.
-	**quote_asset_symbol:** Symbol for the quote asset.
-	**quote_asset_name:** Name of the quote asset.
-	**min_size:** Minimum trade size for the pool, in smallest units of the base asset.
-	**lot_size:** Minimum increment for trades in this pool, in smallest units of the base asset.
-	**tick_size:** Minimum price increment for trades in this pool.

### Response structure

```json
[
	{
	“pool_id”: “string”,
	“pool_name”: “string”,
	“base_asset_id”: “string”,
	“base_asset_decimals”: integer,
	“base_asset_symbol”: “string”,
	“base_asset_name”: “string”,
	“quote_asset_id”: “string”,
	“quote_asset_decimals”: integer,
	“quote_asset_symbol”: “string”,
	“quote_asset_name”: “string”,
	“min_size”: integer,
	“lot_size”: integer,
	“tick_size”: integer
	},
	…
]
```

## /get_24hr_volume/:pool_ids

Retrieves the 24-hour trading volume in the base asset for specified pools.	Delimit `pool_ids` by commas.

### Response

The response is a JSON object where each key is a `pool_id`, and each value is the 24-hour volume (in the smallest units of the base asset) for that pool.

```json
{
	“pool_id_1”: volume,
	“pool_id_2”: volume,
	…
}
```

## /get_historical_volume/:pool_ids/:start_time/:end_time

Use this endpoint to get historical volume for pools for a specific time range. The	`pool_ids` are delimited by commas, `start_time` and `end_time` are in Unix timestamp seconds. 

### Response

Returns the historical volume for each specified pool within the given time range.

```json
{
	“pool_id_1”: total_pool1_volume,
	“pool_id_2”: total_pool2_volume,
	...
}
```

## /get_24hr_volume_by_balance_manager_id/:pool_id/:balance_manager_id

Takes `pool_id` and `balance_manager_id` as input.

### Response

The endpoint returns maker and taker volumes for the last 24 hours.

```json
[ maker_volume, taker_volume ]
```

## /get_historical_volume_by_balance_manager_id/:pool_ids/:balance_manager_id/:start_time/:end_time

Get historical volume by balance manager for a specific time range, where `pool_ids` are delimited by commas and `start_time` and `end_time` in Unix timestamp seconds.

### Response

```json
{
	“pool_id_1”: [maker_volume, taker_volume],
	“pool_id_2”: …
}
```

### Example request

```json
/get_historical_volume_by_balance_manager_id
/0xb663828d6217467c8a1838a03793da896cbe745b150ebd57d82f814ca579fc22
,0xe05dafb5133bcffb8d59f4e12465dc0e9faeaa05e3e342a08fe135800e3e4407
/0x344c2734b1d211bd15212bfb7847c66a3b18803f3f5ab00f5ff6f87b6fe6d27d
/1731260703
/1731692703
```

## /get_historical_volume_by_balance_manager_id_with_interval/:pool_ids/:balance_manager_id/:start_time/:end_time/:interval

Get historical volume by `BalanceManager` for a specific time range with intervals where `pool_ids` are delimited by commas,	`start_time`, `end_time`, and `interval` in Unix timestamp seconds.	If `start_time` is 5, `end_time` is 10, and `interval` is 2, the response includes volume from 5-7 and 7-9 with start time of the periods as keys.

### Response

```json
{
“time_1”: {
		“pool_id_1”: [maker_volume, taker_volume],
		“pool_id_2”: …
	},
“time_2”: {
	“pool_id_1”: [maker_volume, taker_volume],
	“pool_id_2”: …
	}
}
```

### Request

```rust
/get_historical_volume_by_balance_manager_id_with_interval
/0xb663828d6217467c8a1838a03793da896cbe745b150ebd57d82f814ca579fc22,
0xe05dafb5133bcffb8d59f4e12465dc0e9faeaa05e3e342a08fe135800e3e4407
/0x344c2734b1d211bd15212bfb7847c66a3b18803f3f5ab00f5ff6f87b6fe6d27d
/1731460703
/1731692703
/86400
```
