Struct ClosureMetric

pub struct ClosureMetric<F> { /* private fields */ }
Expand description

A Prometheus metric whose value is computed at collection time by the provided closure.

WARNING: The provided closure must be fast (~milliseconds or faster), since it blocks metric collection.
