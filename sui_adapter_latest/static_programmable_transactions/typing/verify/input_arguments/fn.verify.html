Verifies two properties for input objects:

  1. That the Pure inputs can be serialized to the type inferred and that the type is permissible
    • Can be relaxed under certain execution modes
  2. That any Object arguments are used validly. This means mutable references are taken only on mutable objects. And that the gas coin is only taken by value in transfer objects
