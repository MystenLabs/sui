Struct Protobuf 

Source 
pub struct Protobuf<T>(pub T);
Expand description

Transparent wrapper that delegates Encode/Decode to prost::Message::encode / prost::Message::decode.

The inner value is exposed via Deref / DerefMut, the public 0 field, and into_inner. The wrapper carries no runtime overhead beyond the trait dispatch.
