pub fn get_balance( &self, owner: SuiAddress, coin_type: TypeTag, ) -> SuiResult<TotalBalance>

This method first gets the balance from per_coin_type_balance cache. On a cache miss, it gets the balance for passed in coin_type from the all_balance cache. Only on the second cache miss, we go to the database (expensive) and update the cache. Notice that db read is done with spawn_blocking as that is expected to block

Source

pub fn get_all_balance( &self, owner: SuiAddress, ) -> SuiResult<Arc<HashMap<TypeTag, TotalBalance>>>

This method gets the balance for all coin types from the all_balance cache. On a cache miss, we go to the database (expensive) and update the cache. This cache is dual purpose in the sense that it not only serves get_AllBalance() calls but is also used for serving get_Balance() queries. Notice that db read is performed with spawn_blocking as that is expected to block
