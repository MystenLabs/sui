Attribute Macro DefaultConfig

#[DefaultConfig]
Expand description

Attribute macro to be applied to config-based structs. It ensures that the struct derives serde traits, and Debug, that all fields are renamed with “kebab case”, and adds a #[serde(default = ...)] implementation for each field that ensures that if the field is not present during deserialization, it is replaced with its default value, from the Default implementation for the config struct.
