Expand description

Initialize a forked network by fetching the fork checkpoint from the remote endpoint when needed, applying seed metadata, and starting a local Simulacrum instance from the highest checkpoint already persisted locally. Also builds the checkpoint subscription broker; the returned handle must be passed to run so the gRPC server exposes the streaming RPC.

data_dir is the root folder where the fork state is persisted. If None, a default path is used. See the [FileSystemStore] docs for details.
