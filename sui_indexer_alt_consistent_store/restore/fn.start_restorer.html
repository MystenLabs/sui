Expand description

Set-up and run the Restorer, using the provided arguments (expected to be extracted from the command-line). The service will run until restoration complete.

path is the path to the RocksDB database to restore into. It will be created if it does not exist. formal_snapshot_args describes the formal snapshot source, connection_args controls how to connect to it, restore_args controls the restoration process itself, and config configures the RocksDB database.
