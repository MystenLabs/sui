Expand description

The response from processing a transaction or a certified transaction

Fields§

§status: SuiExecutionStatus

The status of the execution

§executed_epoch: EpochId

The epoch when this transaction was executed.

§gas_used: GasCostSummary§modified_at_versions: Vec<SuiTransactionBlockEffectsModifiedAtVersions>

The version that every modified (mutated or deleted) object had before it was modified by this transaction.

§shared_objects: Vec<SuiObjectRef>

The object references of the shared objects used in this transaction. Empty if no shared objects were used.

§transaction_digest: TransactionDigest

The transaction digest

§created: Vec<OwnedObjectRef>

ObjectRef and owner of new objects created.

§mutated: Vec<OwnedObjectRef>

ObjectRef and owner of mutated objects, including gas object.

§unwrapped: Vec<OwnedObjectRef>

ObjectRef and owner of objects that are unwrapped in this transaction. Unwrapped objects are objects that were wrapped into other objects in the past, and just got extracted out.

§deleted: Vec<SuiObjectRef>

Object Refs of objects now deleted (the old refs).

§unwrapped_then_deleted: Vec<SuiObjectRef>

Object refs of objects previously wrapped in other objects but now deleted.

§wrapped: Vec<SuiObjectRef>

Object refs of objects now wrapped in other objects.

§accumulator_events: Vec<SuiAccumulatorEvent>§gas_object: OwnedObjectRef

The updated gas object reference. Have a dedicated field for convenient access. It’s also included in mutated.

§events_digest: Option<TransactionEventsDigest>

The digest of the events emitted during execution, can be None if the transaction does not emit any event.

§dependencies: Vec<TransactionDigest>

The set of transaction digests this transaction depends on.

§abort_error: Option<SuiMoveAbort>

The abort error populated if the transaction failed with an abort code.
