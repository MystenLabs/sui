Return the latest protocol version that is not newer than the requested version (or Err if there is no such version).

The returned [ProtocolVersion] is the protocol version that introduced the returned SystemPackagesVersion; this may be older than the requested version if either:

  1. the system packages did not change when version was released, or
  2. this binary is older than the requested version and therefore doesnâ€™t know about the latest version of the system packages

You can distinguish these cases by comparing version with [ProtocolVersion::MAX].
