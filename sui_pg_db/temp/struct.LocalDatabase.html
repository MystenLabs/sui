pub fn new(dir: PathBuf, port: u16) -> Result<Self>

Start a local postgres database service.

dir: The location of the on-disk postgres database. The database must already exist at the provided path. If you instead want to create a new database see Self::new_initdb.

port: The port to listen for incoming connection on.
