Compact representation of the replay cache for serialization. Contains the execution context (epoch_id from transaction effects, checkpoint from transaction info) and a list of all objects accessed during replay, with their type information but without object content. The epoch_id represents the epoch in which the original transaction was executed. The checkpoint represents the checkpoint sequence number where the transaction was included.
