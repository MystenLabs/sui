Expand description

A TransactionStore has to be able to retrieve transaction data for a given digest. To replay a transaction the data provided to sui_execution::executor::Executor::execute_transaction_to_effects must be available. Some of that data is not provided by the user. It is naturally available at runtime on a live system and later saved in effects and in the context of a checkpoint.
