pub fn transaction_filter_to_query(
    filter: &TransactionFilter,
    max_literals: usize,
) -> Result<BitmapQuery, RpcError>
Expand description

Convert a proto TransactionFilter DNF filter into a BitmapQuery.

max_literals caps the total number of literals across all terms. Each literal can become one bitmap dimension stream, so this prevents one filter from monopolizing bitmap scan fanout.
