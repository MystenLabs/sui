§read_replica: Option<Arc<dyn RpcStateReader + Send + Sync>>

If is_simulator is true, the executor will be a Simulacrum, and this will be a RpcStateReader that can be used to spawn the equivalent of a fullnode rest api. This can then be used to serve an indexer that reads from said rest api service.

§offchain_config: Option<OffChainConfig>

Configuration for offchain state reader read from the file itself, and can be passed to the specific indexing and reader flavor.

§offchain_reader: Option<Box<dyn OffchainStateReader>>

A trait encapsulating methods to interact with offchain state.
