pub struct ObjectInner {
    pub data: Data,
    pub owner: Owner,
    pub previous_transaction: TransactionDigest,
    pub storage_rebate: u64,
}

Fields§

§data: Data

The meat of the object

§owner: Owner

The owner that unlocks this object

§previous_transaction: TransactionDigest

The digest of the transaction that created or last mutated this object

§storage_rebate: u64

The amount of SUI we would rebate if this object gets deleted. This number is re-calculated each time the object is mutated based on the present storage gas price.

pub fn object_size_for_gas_metering(&self) -> usize

Approximate size of the object in bytes. This is used for gas metering. This will be slightly different from the serialized size, but we also don’t want to serialize the object just to get the size. This approximation should be good enough for gas metering.
